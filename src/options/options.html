YouTube Summary with ChatGPT

Extension Settings

OpenAI API Configuration

To use the inline summary feature, you need to provide your OpenAI API key. You can get one from OpenAI's platform.

Usage Information

How it works:

  1. Navigate to any YouTube video
  2. Click the "View AI Summary" button in the sidebar widget
  3. The summary will appear directly on the YouTube page
  4. No need to open ChatGPT in a new tab!

Important Notes:

  • Your API key is stored locally in your browser and never sent anywhere except OpenAI
  • Using the API will incur costs on your OpenAI account
  • GPT-5 Mini is recommended for cost-effectiveness
  • GPT-5 provides the most advanced summaries with better context understanding
  • Max tokens are automatically calculated based on transcript length
  • Keep your API key secure and don't share it with others

Fallback Mode

If you don't have an API key, the extension will continue to work in fallback mode by opening ChatGPT in a new tab.
